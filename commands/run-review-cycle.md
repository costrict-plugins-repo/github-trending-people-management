# /run-review-cycle
Run a fair, consistent review cycle.
## Steps
1. **Set the bar** — Anchor on level expectations using `career-ladder` skill.
2. **Write reviews** — Draft evidence-based reviews using `performance-review` skill.
3. **Prepare deliveries** — Plan how news is delivered using `difficult-conversation` skill.
4. **Plan growth** — Turn each review into next steps using `growth-plan` skill.
## Output
A review-cycle pack: calibrated bar, drafted reviews, delivery plans, and a growth plan per person.
Consider following up with `/prep-one-on-one` to carry actions into ongoing conversations.
